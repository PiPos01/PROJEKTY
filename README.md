## Project 1: Fire Damage Classification (Portugal, Summer 2018)
**Tools:** Satellite Imagery, Remote Sensing, ΔNBR Index

This project focuses on assessing the environmental impact of the devastating wildfires in Portugal during the summer of 2018. 

* **Objective:** To classify the burn severity of the affected areas.
* **Methodology:** Calculation of the **ΔNBR (Differenced Normalized Burn Ratio)** index using pre-fire and post-fire satellite data.
* **Outcome:** A detailed damage classification map distinguishing between unburned areas, low-severity, and high-severity burn zones.
* **File:** `dNBR_layout.pdf` (Professional map layout included in the repo).

## Project 3: Pharmacy Locator & Routing App
**Tools:** ArcGIS Experience Builder

A user-friendly, interactive web application designed to help patients not only locate the nearest pharmacies but also navigate to them efficiently.

* **Key Features:**
    * **Interactive Map & Pop-ups:** Precise pharmacy locations featuring detailed pop-ups with essential information (e.g., name, address, opening hours).
    * **"Near Me" Functionality:** A dedicated tool allowing users to easily find the closest pharmacies within a specific radius of their current location.
    * **Directions & Routing:** Integrated navigation widgets that generate optimal routes for patients to reach their selected pharmacy.
    * **Search Tool:** Quick address and location search capabilities.
    * **Mobile-Responsive Design:** Built with **ArcGIS Experience Builder**, ensuring smooth operation across desktop and mobile devices.
